# Mi Primer Repositorio
## Descripción
Este proyecto contiene archivos CSV que simulan la información de un sistema básico de ventas. Incluye datos de clientes y ventas, los cuales pueden usarse para pruebas, análisis o ejemplos educativos.

# Archivos Incluidos
## ventas.csv
### Contiene el registro de ventas realizadas.
- Columnas principales:
- fecha
- producto
- cantidad
- precio_unitario
- total
- id_cliente (relacionado con clientes.csv)

## clientes.csv
### Contiene la información de los clientes del sistema.
Columnas principales:
- id_cliente
- nombre
- apellido
- correo
- telefono
# Relación entre archivos
El campo de ventas.csv corresponde al campo de clientes.csv, lo que permite identificar qué cliente realizó cada compra.
